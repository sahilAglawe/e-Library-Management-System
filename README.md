📚 E-Library Management System

A complete web-based library automation system built using Java, JSP, Servlets, and MySQL.
This project helps institutions manage books, students, issuing, returning, authentication, and administrative operations in a modern, digital, and user-friendly way.

🚀 Features

🔐 #User Authentication

Admin login
Librarian login
Secure session handling


📘 Book Management

Add new books
View / update / delete books
Search books
Category-wise book arrangement


🧑‍🎓 Student Management

Add students
Update student details
Maintain complete student records


🔄 Issue & Return Module

Issue books to students
Return books with date tracking
Automatic status update
Full record history of issued & returned books


🧾 Real-Time Records

View all issued books
View all returned books
Track book availability
Error/Success pop-up messages


🎨 UI & UX

The system includes a modern, responsive, Bootstrap-based interface:
Clean dashboard layout
Professional-themed login cards
Organized tables for quick insights
Popup alerts for smoother interactions


🏗️ Tech Stack

Layer -	Technology
Frontend -	JSP, HTML5, CSS3, Bootstrap
Backend	Java - Servlets, JDBC
Database -	MySQL
Server -	Apache Tomcat



🎯 Purpose

This project aims to modernize traditional library operations by providing:
Fast retrieval of book details
Error-free issuing & returning process
Time-saving automation
Centralized record management
Perfect for colleges, schools, and small institutions.


📥 How to Run

Clone the repository
Import into IntelliJ / Eclipse
Configure Tomcat Server
Create MySQL database → import tables
Run the project
